# Joseph Wilson - Musician Portfolio Website

## 1. Project Description
A personal portfolio website for Joseph Wilson, a bass musician/DJ. The site showcases his music, services, biography, and allows fans/clients to get in touch. It features a dark, cinematic aesthetic with teal/cyan accent colors.

## 2. Page Structure
- `/` - Home (hero, listen now, about preview, services preview, masterclass, moments, testimonials, contact)
- `/about` - About Me
- `/services` - Services
- `/contact` - Contact
- `/login` - Login
- `/signup` - Signup

## 3. Core Features
- [x] Login page
- [x] Signup page
- [x] Home page (full sections)
- [ ] About Me page
- [ ] Services page
- [ ] Contact page with map and form

## 4. Data Model Design
No database needed - static portfolio site.

## 5. Backend / Third-party Integration Plan
